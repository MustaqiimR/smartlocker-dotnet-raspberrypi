PROJECT OVERVIEW
Objectives, Scope & Technology
Objective
Build a comprehensive locker management system for on-premise deployment on Raspberry Pi
Support item borrowing, locker access control, and QR unlock functionality
Provide role-based web interfaces for Admin and Staff management
Scope
Item inventory management with categorization
Request approval workflow with QR token generation
Borrow/return lifecycle with overdue tracking
Audit logging and system monitoring
Technology Stack
Backend: ASP.NET Core 8.0
Frontend: Razor Pages with Bootstrap
Database: SQLite with Entity Framework Core
Deployment: Raspberry Pi OS 64-bit
Deployment
Local network touchscreen kiosk interface
Admin web portal for system management
Staff web portal for requests and borrows
Systemd service for auto-start on boot
